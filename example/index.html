Composant Access Settings

Installation

CDN

La façon la plus simple et la plus rapide. 
<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>My accessible page</title>
    <script src="https://cdn.jsdelivr.net/npm/access-settings"></script>
  </head>
  <body>
    <access-settings all></access-settings>
  </body>
</html>

NPM

npm install access-settings
<script src="node_modules/access-settings/dist/index.js"></script>

Téléchargement direct

Vous pouvez télécharger directement le fichier sur GitHub.

Utilisation

all

Pour afficher toutes les options, utiliser l'attribut all.

<access-settings all></access-settings>

dyslexic-font

Pour activer l'option police dyslexie, utiliser l'attribut dyslexic-font.

La police utilisée est Open Dyslexic depuis le serveur CDN|Fonts.

<access-settings dyslexic-font>
</access-settings>

invert-colors

Pour activer l'option d'inversion des couleurs, utiliser l'attribut invert-colors.

<access-settings dyslexic-font invert-colors>
</access-settings>

contrast

Pour activer le réglage du contraste, utiliser l'attribut contrast.

<access-settings dyslexic-font invert-colors contrast>
</access-settings>

font-size

Pour activer le réglage de la taille du texte, utiliser l'attribut font-size.

<access-settings dyslexic-font invert-colors contrast font-size>
</access-settings>

line-height

Pour activer le réglage de la hauteur de ligne, utiliser l'attribut line-height.

<access-settings dyslexic-font invert-colors contrast font-size line-height>
</access-settings>

side="left"

Par défaut le composant est placé sur le bord droit de l'écran. Pour le positionner à gauche, utiliser l'attribut side="left" 

<access-settings all side="left">
</access-settings>

rounded

Pour que le contour de l'icône soit arrondi, utiliser l'attribut rounded

<access-settings all side="left" rounded>

lang

La langue est déterminée par l'attribut lang de l'élément racine html.

Les langues disponibles sont fr et en.
